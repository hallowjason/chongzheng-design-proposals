Design System 提案

三種視覺風格，請團隊選出偏好方向

Proposal A

提案 A — 崇正原色系 × Apple Token 架構

  • 沉穩有根的米沙色底配深木炭側欄、青色強調
  • Apple 圓潤陰影、最完整的 primitive/semantic 雙層 token
  • 最符合道場氛圍，精緻而不張揚
Proposal B

提案 B — 暖中性 × 文件感設計

  • 暖米白底配深棕側欄、Notion 卡片塊狀結構
  • 三份中最寬鬆行高、最舒適的閱讀節奏
  • 溫和親切，如紙張般的長時操作體驗
Proposal C

提案 C — Stripe Premium × 數據驅動

  • 純白底配深海軍藍側欄、招牌 Stats Strip 統計卡
  • 最高資訊密度，適合進階管理員
  • 現代專業科技感，四色邊框數字卡最為突出
